Want to mint?

Here is what we know so far:

  • Either you mint in ronin or abstract, each network has its own supply. See Minting supply
  • Mining in ronin would use Mavis Launchpad, abstract would use Openseas.
  • Holding the highest tier of NFT in each games under ronen have a guaranteed spot. See GTD Allocation
  • Date is to be announced.
  • Mint Price is not final, currently its set to (5 RON + 2,000 RONEN / WETH + GRONKE equivalent).
  • There are 3 phases:
    Phase 1 is for NFT holder, which will have to pre-pay 2,000 RONEN/GRONKE.
    Phase 2 is for RP Holders, user can use Ronen Points + 2,000 RONEN to secure GTD spots.
    Phase 3 is for Collab winners, like winners from giveaways or collab campaigns, must also pay 2,000 RONEN.
